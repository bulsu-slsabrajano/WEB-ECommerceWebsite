# WEB-ECommerceWebsite
**Project Description**

Vanguard’s Delights is an eCommerce website focused on selling pastry-based products such as cakes, bread, and other baked goods. The system is designed to provide users with a convenient and user-friendly platform where they can browse products, place orders, and manage their accounts online.

The website also includes an administrative side that allows authorized users to manage products and monitor the system efficiently. Overall, the project aims to simulate a real-world online pastry shop that improves accessibility and simplifies the ordering process for customers.

**Features**

The system includes several core features that make it functional and user-friendly:
**Login and Signup UI** - Users can create an account and log in securely to access the system.
**Homepage** - Displays featured products, promotions, and general information about Vanguard’s Delights.
**Order Page** - Allows users to browse available pastries, select items, and place orders.
**User Profile Management **- Users can update their personal information such as username, password, and other details.
**Admin Panel** - The admin has full control over the system, including:
* Adding new products
* Updating existing products
* Deleting products
**Ecommerce Functionalities** - Includes basic online shopping features like product listings, ordering system, and user account handling.

**Technologies Used**

The system was developed using the following technologies:
**MySQL / phpMyAdmin** - Used as the database to store user information, product details, and order data.
**Visual Studio Code** - Used as the main code editor for developing the backend and frontend of the system.
**PHP** - Used as the server-side scripting language to handle system logic, database interaction, and user authentication.
**HTML, CSS, JavaScript** - Used for designing the user interface and improving user experience.

**Setup Installation Guide**
Steps to Install the System:
**1. Install Required Software**
- Download and install XAMPP or any similar local server environment.
**2. Set Up the Project Folder**
- Place the Vanguard’s Delights project folder inside the htdocs directory of XAMPP.
**3. Start the Server**
- Open XAMPP Control Panel.
- Start Apache and MySQL.
**4. Import the Database**
- Open phpMyAdmin in your browser (http://localhost/phpmyadmin).
- Create a new database (e.g., vanguards_delights_db).
- Import the provided .sql file into the database.
**5. Configure Database Connection**
- Open the project files in Visual Studio Code.
- Locate the database connection file (e.g., connection.php).
- Update the database name, username, and password if needed.
**6. Run the Project**
- Open a browser and go to:
- http://localhost/your-project-folder
- The system should now be running.
